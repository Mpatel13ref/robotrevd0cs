# AI Vision Sensor — Documentation

VEX V5 AI Vision Sensor documentation site for the Robot Revolution.

**Live site:** `https://<your-username>.github.io/<repo-name>/`

## Structure

```
index.html       ← main page
assets/
  style.css      ← all styles
  script.js      ← nav highlighting + copy button
_config.yml      ← GitHub Pages config
```

## Deploying to GitHub Pages

1. Create a new repository on GitHub (e.g. `robot-revolution-docs`)
2. Upload all files in this folder to the repo root
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose **main** branch, **/ (root)** folder — click Save
6. Your site will be live at `https://<your-username>.github.io/robot-revolution-docs/` within a minute or two

## Adding Content

The three placeholder sections (Application, Useful Functions, Full API) are ready to be filled in. Each one is a `<section>` in `index.html` — replace the `.placeholder` div with your content.
