01
Getting Started

Initialization

Initialization of the AI Vision Sensor is straightforward. Declare it wherever you initialize your motors — the same scope, same location.

C++ 
// Replace PORT1 with the port your AI Vision is plugged into
aivision AiVision = aivision(PORT1);

Declare the aivision object at the top of your project alongside all other device declarations for consistent, readable code.

02
Setup

Configuring

Three environments let you configure the sensor. Each has a different capability level.

Option A
Brain

Go to Devices and tap the sensor. Toggle between Image mode (rectangles on a graph) and Objects mode (live picture).

Alternative
Option B
VS Code

Simple and effective — capture screenshots directly from the sensor while plugged in. Great for quick checks and verification.

Recommended
Option C
VEXcode

Most intuitive. Live video feed so you see exactly what the sensor sees in real time.

Live Video

From the Brain

Display modes

  • Image mode — Graph view. Draws rectangles around detected objects as they appear.
  • Objects mode — Shows a live picture from the sensor's current perspective.

Looking for April Tags

  • Tap "Object Color" repeatedly until it reads "Tags."
  • Move the sensor around — it now detects only the April Tags pretrained on the device.

From VS Code

In VS Code, capture a screenshot directly from the sensor while it is plugged into the computer. Quick and straightforward for verification purposes.

From VEXcode

  • Provides a live video feed directly from the sensor.
  • Most straightforward method for setting up and verifying object detection signatures.
